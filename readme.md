```Instruction to use the App```

1) Start inserting obstacles by clicking on the tiles. Second click on the selected tiles removes the obstacle. 
2) Click on 'Select Start Cell' button to select starting cell. Second click on the selected tile will remove the selection. 
3) Click on 'End Start Cell' button to select ending cell. Second click on the selected tile will remove the selection. 
4) Click on 'Find Path' button to find the path between start and end cell. 
5) Click on 'Clear Map' to start over again.
